## Study Tips

### Week-by-Week Plan

**Week 1-2: Identity, Governance, Monitoring**
- Complete Microsoft Learn modules
- Practice Conditional Access policies
- Set up Azure Policy examples
- Configure Log Analytics workspace

**Week 3-4: Data Storage**
- Deploy SQL Database with geo-replication
- Create Cosmos DB with multiple regions
- Practice Data Factory pipelines
- Implement storage lifecycle policies

**Week 5: Business Continuity**
- Configure Azure Backup
- Set up Azure Site Recovery
- Test failover scenarios
- Deploy across availability zones

**Week 6-7: Infrastructure**
- Deploy multi-tier application
- Configure VNet peering
- Set up VPN Gateway
- Deploy AKS cluster

**Week 8: Review and Practice**
- Take practice exams
- Review weak areas
- Study architecture diagrams
- Read case studies

## Exam Day Tips

✅ Get good sleep the night before  
✅ Arrive 15 minutes early  
✅ Read questions carefully  
✅ Manage your time (120 minutes)  
✅ Mark questions for review  
✅ Don't overthink - trust your preparation  
✅ Use process of elimination  
✅ Consider cost, security, and scalability

## Certification Path

**Prerequisites:**
- AZ-104: Azure Administrator Associate (recommended)
- 1-2 years of Azure experience

**After AZ-305:**
- Maintain certification with renewal
- Consider specialty certifications:
  - AZ-400: DevOps Engineer Expert
  - AZ-500: Security Engineer Associate
  - AZ-700: Network Engineer Associate
