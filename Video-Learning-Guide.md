# Video Learning Resources

## 🎥 Most Reliable Video Sources

### Official Microsoft Resources (Always Available)

**Microsoft Learn - Interactive Training**
- URL: https://learn.microsoft.com/training/courses/az-305t00
- Free, official, always updated
- Interactive labs with sandbox

**Microsoft Documentation**
- URL: https://docs.microsoft.com/azure
- Official docs with embedded videos

**Azure Architecture Center**
- URL: https://learn.microsoft.com/azure/architecture
- Reference architectures with explanations

### YouTube - Search These Exact Terms

**For Complete Courses:**
1. Search: **"John Savill AZ-305"** - Look for 20+ hour video
2. Search: **"Adam Marczak Azure"** - Look for playlists
3. Search: **"FreeCodeCamp Azure"** - Look for 10+ hour courses
4. Search: **"AZ-305 complete course"** - Filter by: This year

**For Specific Topics:**
- "Azure AKS tutorial"
- "Azure Firewall complete guide"
- "Cosmos DB tutorial"
- "Azure networking fundamentals"
- "Azure security best practices"

### How to Find Working Videos

1. Go to **youtube.com**
2. Search the term above
3. Click **Filters** → **Upload date** → **This year**
4. Sort by **View count**
5. Look for videos with 50K+ views

### Recommended Channels (Search by Name)

- **John Savill** (NTFAQGuy) - Most comprehensive

## 📺 Alternative Learning

- Free certification exam voucher

**Microsoft Reactor** (Free Events)
- URL: https://developer.microsoft.com/reactor
- Live coding sessions
- Q&A with experts

**Azure Friday** (Weekly Show)
- URL: https://learn.microsoft.com/shows/azure-friday
- Short 10-15 min episodes
- New Azure features

Instead of relying on specific links that may break:
1. Subscribe to the channels above
2. Enable notifications
3. Create your own playlists
4. Download important videos for offline viewing
