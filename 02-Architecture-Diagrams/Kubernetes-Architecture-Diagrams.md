## 4. Kubernetes Service Types

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Service Types                                     │
│                                                                      │
│  1. ClusterIP (Default - Internal only)                             │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │  Service: my-service (10.0.1.100:80)                       │     │
│  │              │                                              │     │
│  │              ├──► Pod1 (10.244.1.5:8080)                   │     │
│  │              ├──► Pod2 (10.244.1.6:8080)                   │     │
│  │              └──► Pod3 (10.244.2.7:8080)                   │     │
│  │                                                             │     │
│  │  Access: Only within cluster                               │     │
│  └────────────────────────────────────────────────────────────┘     │
│                                                                      │
│  2. NodePort (External access via Node IP)                          │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │  Service: my-service (NodePort: 30080)                     │     │
│  │                                                             │     │
│  │  Node1:30080 ──┐                                           │     │
│  │  Node2:30080 ──┼──► Service ──► Pods                       │     │
│  │  Node3:30080 ──┘                                           │     │
│  │                                                             │     │
│  │  Access: <NodeIP>:30080                                    │     │
│  └────────────────────────────────────────────────────────────┘     │
│                                                                      │
│  3. LoadBalancer (Cloud provider LB)                                │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │  External IP: 52.123.45.67                                 │     │
│  │              │                                              │     │
│  │              ▼                                              │     │
│  │  Azure Load Balancer                                       │     │
│  │              │                                              │     │
│  │              ├──► Node1:30080 ──┐                          │     │
│  │              ├──► Node2:30080 ──┼──► Service ──► Pods      │     │
│  │              └──► Node3:30080 ──┘                          │     │
│  │                                                             │     │
│  │  Access: 52.123.45.67:80                                   │     │
│  └────────────────────────────────────────────────────────────┘     │
│                                                                      │
│  4. ExternalName (DNS CNAME)                                        │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │  Service: external-db                                      │     │
│  │              │                                              │     │
│  │              └──► CNAME: db.example.com                    │     │
│  │                                                             │     │
│  │  Access: Pods use "external-db" → resolves to external DNS │     │
│  └────────────────────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────────────────────┘
```

## 5. Kubernetes Storage

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Storage Architecture                              │
│                                                                      │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │              Storage Class                                  │     │
│  │  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐     │     │
│  │  │   Standard   │  │   Premium    │  │   Managed    │     │     │
│  │  │   (HDD)      │  │   (SSD)      │  │   (Azure)    │     │     │
│  │  └──────┬───────┘  └──────┬───────┘  └──────┬───────┘     │     │
│  └─────────┼──────────────────┼──────────────────┼─────────────┘     │
│            │                  │                  │                   │
│            ▼                  ▼                  ▼                   │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │         Persistent Volume (PV) - Cluster Resource          │     │
│  │  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐     │     │
│  │  │   PV-001     │  │   PV-002     │  │   PV-003     │     │     │
│  │  │   100GB      │  │   50GB       │  │   200GB      │     │     │
│  │  │   Available  │  │   Bound      │  │   Available  │     │     │
│  │  └──────────────┘  └──────┬───────┘  └──────────────┘     │     │
│  └─────────────────────────────┼────────────────────────────────┘     │
│                                │ Binding                             │
│                                ▼                                     │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │    Persistent Volume Claim (PVC) - Namespace Resource      │     │
│  │  ┌──────────────────────────────────────────────────┐      │     │
│  │  │  PVC: my-pvc                                     │      │     │
│  │  │  Request: 50GB, ReadWriteOnce                    │      │     │
│  │  │  Status: Bound to PV-002                         │      │     │
│  │  └────────────────────┬─────────────────────────────┘      │     │
│  └───────────────────────┼────────────────────────────────────┘     │
│                          │ Mount                                    │
│                          ▼                                          │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │                    Pod                                      │     │
│  │  ┌──────────────────────────────────────────────────┐      │     │
│  │  │  Container                                       │      │     │
│  │  │  ┌────────────────────────────────────────┐     │      │     │
│  │  │  │  Volume Mount: /data                   │     │      │     │
│  │  │  │  Source: my-pvc                        │     │      │     │
│  │  │  └────────────────────────────────────────┘     │      │     │
│  │  └──────────────────────────────────────────────────┘      │     │
│  └────────────────────────────────────────────────────────────┘     │
│                                                                      │
│  Volume Types:                                                      │
│  - emptyDir (Pod lifetime)                                          │
│  - hostPath (Node filesystem)                                       │
│  - configMap (Configuration)                                        │
│  - secret (Sensitive data)                                          │
│  - persistentVolumeClaim (Persistent storage)                       │
└─────────────────────────────────────────────────────────────────────┘
```
